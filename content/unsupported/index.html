Unsupported Features and Errata

Unsupported features and errata of the DFDL Specification as of Daffodil 2.7.0 are listed below:

DFDL Features

  • “type” component of DFDL infoset [DAFFODIL-182], [DAFFODIL-1852]
  • “unionMemberSchema” component of DFDL infoset [DAFFODIL-1633]
  • “valid” component of DFDL infoset [DAFFODIL-813]
  • bi-directional text (Note this is being removed from the DFDL Specification v1.0 per Erratum 5.43, but may return as an experimental feature, or in a future version of the DFDL spec.)
  • byte-value entities (aka raw-byte entities)[DAFFODIL-258]
  • defaulting of values for required elements - when unparsing [DAFFODIL-115]
  • extended ICU symbol ‘I’ in calendarPattern [DAFFODIL-1462]
  • floating elements in sequences [DAFFODIL-643]
  • unicodeByteOrderMark (Note this is being removed from the DFDL Specification v1.0 per Erratum 5.50)
  • validation (when unparsing) [DAFFODIL-1582]

XML Schema Features

  • fixed [DAFFODIL-117]
  • default [DAFFODIL-115]

Properties and Property Enumerations

  • binaryFloatRep=”ibm390Hex” [DAFFODIL-244]
  • documentFinalTerminatorCanBeMissing=”yes” [DAFFODIL-230]
  • encodingErrorPolicy=”error” (Note: this is accepted, but behaves as “replace”)[DAFFODIL-935]
  • floating=”yes” [DAFFODIL-643]
  • lengthKind=”endOfParent” [DAFFODIL-238], [DAFFODIL-567]
  • nilKind=”logicalValue” [DAFFODIL-201]
  • occursCountKind=”stopValue”, occursStopValue [DAFFODIL-501]
  • utf16Width=”variable” [DAFFODIL-551]
  • calendarObserveDST [DAFFODIL-521]

Miscellaneous

  • XPath query-style expressions [DAFFODIL-1118]

Errata (Minor Technical Fixes)

Items 5.1 through 5.63 of the DFDL v1.0 Spec Errata are implemented, except for those listed below:

  • Erratum 5.1 - Test for escapeEscapeCharacter in escape block data [DAFFODIL-2421]
  • Erratum 5.3 - dfdl:escapeCharacterPolicy property [DAFFODIL-2415]
  • Erratum 5.6 - “form” attribute (note that “elementFormDefault” is supported) [DAFFODIL-2416]
  • Erratum 5.14 - Check range of binaryDecimalVirtualPoint at runtime [DAFFODIL-2417]
  • Erratum 5.29 - Intersect and except operators, dfdl:checkRangeInclusive, dfdl:checkRangeExclusive functions [DAFFODIL-2379], [DAFFODIL-1515]
  • Erratum 5.30 - Character-level scanning insufficient due to raw byte entities [DAFFODIL-258]
  • Erratum 5.39 - Choice branches that are zero-occurrence arrays are missing. [DAFFODIL-2420]
  • Erratum 5.40 - complex type cannot have hiddenGroupRef as its model group [DAFFODIL-2419]
  • Erratum 5.47 - Encoding/Decoding errors - dfdl:encodingErrorPolicy=”error” not implemented.[DAFFODIL-935]
  • Erratum 5.62 - documentFinalTerminatorCanBeMissing - property is not implemented.[DAFFODIL-230]
